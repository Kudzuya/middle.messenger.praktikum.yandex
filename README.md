Описание Проект мессенджера, созданный с использованием Handlebars для верстки основных экранов и компонентного подхода. Для клиентского рендеринга применен Event Bus паттерн с базовым классом Component.

Запуск проекта

npm install — установка зависимостей, npm run start — сборка и запуск, npm run dev — для запуска в режиме разработки, npm run lint — для запуска проверки eslint, npm run stylelint — для запуска проверки stylelint.
